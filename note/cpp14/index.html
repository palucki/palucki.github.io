Generic lambdas && init capture

Allow to use auto as parameter type

[](const auto& lhs, const auto& rhs) {
   return lhs.length() < rhs.length();
}

Ability to init variable when capturing to lambda

[u{std::move(p)}] () { }

[u = std::move(p)] () { }
